1.1.1.4.1 OTA Service Transport Library Interface

Library Interface

OTA Service Transport layer provides the following interface finctions:

Functions

Name Description

OTA_SERVICE_Transport_initialize

Initialize OTA Transport layer

OTA_SERVICE_Transport_Complete

Validate OTA_COMPLTETE

OTA_SERVICE_Transport_MsgRecv

 Read fragmented image Payload

OTA_SERVICE_Transport_MsgSend

Send command to RNBD

OTA_SERVICE_Transport_RspRecv

Read command response back

OTA_SERVICE_Transport_ackResp

Send acknowledgment after every frame

OTA_SERVICE_Transport_CloseOta

OTA validation status update to RNBD
